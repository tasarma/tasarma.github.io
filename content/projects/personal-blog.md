For a while, I wanted to have my own website, but I didn't want to build everything from scratch. I was looking for something that was ready to use and easy to get started with.

Thanks to [Orhun Parmaksız](https://blog.orhun.dev/) that introduced me to [Zola](https://www.getzola.org/), a static site generator written in Rust, and it seemed like the perfect fit.

# Apollo
Zola offers [many themes](https://www.getzola.org/themes/) that you can choose. I decided to go with [Apollo](https://www.getzola.org/themes/apollo/) because I like its minimalistic and modern design.

# How to Set It Up
To set everything up, I followed [this blog post](https://blog.orhun.dev/setting-up-this-blog/) by Orhun Parmaksız along with the Apollo documantation. Everything I needed was already covered there.
