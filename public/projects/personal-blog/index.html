For a while, I wanted to have my own website, but I didn't want to build everything from scratch. I was looking for something that was ready to use and easy to get started with.

Thanks to Orhun Parmaksız that introduced me to Zola, a static site generator written in Rust, and it seemed like the perfect fit.

Apollo

Zola offers many themes that you can choose. I decided to go with Apollo because I like its minimalistic and modern design.

How to Set It Up

To set everything up, I followed this blog post by Orhun Parmaksız along with the Apollo documantation. Everything I needed was already covered there.
